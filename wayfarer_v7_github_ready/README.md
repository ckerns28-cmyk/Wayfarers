# Wayfarer v7 GitHub Ready

A compact browser-first MMO prototype package, designed as a stable GitHub starting point.

## Included
- Mirror Pond region
- Eastern and Western farm fields
- Wolf territories
- Turkey pasture
- Basic combat loop
- Loot and inventory
- Day/night tint
- Edrin Vale NPC with dialogue

## Run locally
```bash
cd worker
npm install
npm run dev
```
Then open the local URL Wrangler shows.
